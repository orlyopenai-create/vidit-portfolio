**1. `components/hero/StatCountUp.tsx`** — 'use client' leaf component

- Add `'use client'` directive at top
- Import `useRef`, `useEffect`, `useState` from React
- Import `useInView` from `'framer-motion'`
- Import `type { Stat }` from `'@/lib/types'`
- Props: `{ stat: Stat }`
- Create a `ref` with `useRef<HTMLDivElement>(null)`
- `const isInView = useInView(ref, { once: true })` — triggers once on viewport entry
- `const [count, setCount] = useState(0)` — starts at zero
- `useEffect` gated on `isInView`:
  - `if (!isInView) return` — do NOT animate until in view
  - `const duration = 1500` (1.5s locked)
  - `const start = performance.now()`
  - `const end = stat.value`
  - RAF loop `tick(now)`: `elapsed = now - start`, `progress = Math.min(elapsed / duration, 1)`, `eased = 1 - Math.pow(1 - progress, 3)` (cubic ease-out), `setCount(parseFloat((eased * end).toFixed(stat.decimals ?? 0)))`, recurse if `progress < 1`
  - Dependencies: `[isInView, stat.value, stat.decimals]`
- Render:
  ```
  <div ref={ref} className="flex flex-col items-center gap-1">
    <span className="font-mono text-3xl md:text-4xl text-accent">
      {stat.prefix}{count.toFixed(stat.decimals ?? 0)}{stat.suffix}
    </span>
    <span className="font-body text-xs text-muted text-center">
      {stat.label}
    </span>
  </div>
  ```
- Export as named: `export function StatCountUp`
